# livecontainer-webredirect-userscript
[日本語版はこちら](README.ja.md)

Userscript files for most major social media apps that when opened from the Safari browser, automatically redirects to the appropriate app within Livecontainer.

# Prerequisites
 - an iPhone (preferably with late iOS version)
 - [Userscripts](https://apps.apple.com/us/app/userscripts/id1463298887) app installed
# How to set it up
Open the Files app. Go to [On My iPhone → Userscripts] place the .js files in your Userscripts library. 
Open safari. Create a new tab, and open the Userscripts Extension. You should see your configuration files you just placed. If you don't, try to refresh by tapping on the refresh icon on the top right of the menu.
Once you have it set up, you can now redirect social media websites on your modded Livecontainer apps.
simple as that.
