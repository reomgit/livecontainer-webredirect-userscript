# livecontainer-webredirect-userscript
[English version](README.md)

Safariブラウザから開いたときに、主要なソーシャルメディアサイトをLiveContainer内の対応アプリへ自動リダイレクトするためのUserscriptファイルです。

# 前提条件
- iPhone（できれば新しめのiOSバージョン）
- [Userscripts](https://apps.apple.com/us/app/userscripts/id1463298887) アプリがインストールされていること

# セットアップ方法
ファイルアプリを開き、[このiPhone内 → Userscripts] に移動して、`.js` ファイルをUserscriptsライブラリに配置します。  
Safariを開いて新しいタブを作成し、Userscripts拡張機能を開きます。先ほど配置した設定ファイルが表示されるはずです。表示されない場合は、メニュー右上の更新アイコンをタップして再読み込みしてください。  
セットアップが完了したら、改造済みLiveContainerアプリでソーシャルメディアのWebサイトをリダイレクトできるようになります。  
以上です。
